Bem-vindo à Guild, Herói.

A SkillForge Guild é um sistema distribuído onde cada developer expõe suas habilidades como um nó vivo. Quando você entra, o sistema recalcula o que a guilda consegue resolver. Quando você evolui, novas possibilidades se desbloqueiam para todos.

  1. Edite manifest.json com suas skills reais — veja o SKILL_MANIFEST_GUIDE
  2. Reinicie o servidor para carregar o manifesto atualizado
  3. Escolha sua primeira quest em Quests
  4. Abra uma issue no repositório com o label hero para se registrar na guilda

Mais informação é revelada conforme você completa quests e sobe de nível. Você está em APPRENTICE — próximo nível: Journeyman (nível 3).
